# **🔤 HTML 101: Building the Structure**

**HTML = HyperText Markup Language**\
It tells the browser **what elements** to display: text, headings, images, links, forms, etc.

**🧱 Example:**

<!DOCTYPE html>

<html>

`  `<head>

`    `<title>My First Page</title>

`  `</head>

`  `<body>

`    `<h1>Hello, World!</h1>

`    `<p>This is my first webpage.</p>

`    `<a href="https://example.com">Visit a website</a>

`  `</body>

</html>

**🧠 Key HTML Tags**

|**Tag**|**What It Does**|
| :- | :- |
|<h1> - <h6>|Headings (big to small)|
|<p>|Paragraph|
|<a>|Link|
|<img>|Image|
|<ul> / <li>|Unordered list items|
|<div>|Container block|
